# Antojo Casero

**Hecho en casa, con amor**

Proyecto React de ecommerce rebrandizado para la marca **Antojo Casero**, orientado a productos artesanales, comida hecha en casa y una experiencia cálida de compra.

## Tecnología

- React 18 con Create React App (`react-scripts`).
- React Router DOM para rutas internas.
- Bootstrap 4/5 y CSS global en `src/App.css`.
- Componentes de UI y carruseles con `react-slick`, `react-awesome-reveal` y dependencias visuales existentes.

## Instalación y uso

```bash
npm install
npm start
```

## Build de producción

```bash
npm run build
```

## Pruebas

```bash
npm test -- --watchAll=false
```

## Rebranding

La primera fase de rebranding centraliza la paleta de Antojo Casero en `src/App.css`, actualiza el header, navegación, footer, metadatos, textos principales, estados visuales y el logo en `public/images/antojo-casero-logo.svg`.

Consulta `docs/REBRANDING_ANTOJO_CASERO.md` para el detalle de auditoría, cambios y pendientes.
